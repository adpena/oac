# Vision

Open Agent Capsule is a practical bet on a simple idea:

**an agent should be portable because its durable intelligence is inspectable, typed, and owned — not because you copied a fragile runtime image.**

The long-term shape is:

- one canonical capsule
- many native projections
- explicit ownership boundaries
- typed candidate bundles between native work and canonical promotion
- explicit proposal records with diffs and provenance
- eval-gated, reversible promotion
- deterministic snapshot and release seams
- optional derived indexes and learned sidecars
- governed learning instead of uncontrolled drift

The near-term shape is intentionally smaller:

- validate the format
- prove useful projections
- prove useful ingest paths
- prove explicit proposal records
- prove a structural eval gate
- prove reversible promotion
- prove deterministic local snapshots

The goal is not to win every abstraction fight up front.
The goal is to build the thinnest real portability layer that can grow into a larger system without collapsing under its own ambition.
